================================================================================
                        TROUBLESHOOTING
================================================================================

Issue: Setup fails
  → Delete venv: rm -rf venv && ./setup.sh

Issue: No search results
  → Check stats: ./run-cli.sh stats
  → Try keyword search: ./run-cli.sh search "term" --method keyword

Issue: Ollama connection error
  → Start Ollama: ollama serve
  → Pull models: ollama pull nomic-embed-text mistral

Issue: Port 8000 in use
  → Use different port: ./run-skill.sh 9000

Issue: API not responding
  → Check health: curl http://localhost:8000/health

See README.md or INTEGRATION.md for more troubleshooting.
